Job Description

Create educational software that is useable by students around the world. Our programming languages are intended to make it fun for students of various skill levels to create video games, apps, and websites. Help us build the languages that inspire students to become life-long coders.

If you are interested in volunteering to code with us, MetaCoders will provide you all the training and documentation you need to create fun educational software for K-12 students. Send us an email below to join our team!
